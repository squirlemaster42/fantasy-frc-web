# Development Setup

> Last updated: 2026-04-12

## Setup Guide

### Prerequisites

- Go 1.24+
- PostgreSQL database
- Tailwind CSS CLI (automatically downloaded)

### Installation

1. Clone the repository
2. Install dependencies: `go mod download`
3. Copy `.env.example` to `.env` and configure values
4. Run migrations: `make migrate`
